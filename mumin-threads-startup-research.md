## 3. Shipping Cost Analysis

### Domestic (India)
- Courier: ₹40-80 per kg (Delhivery, FedEx)
- For 1kg package (3-4 items): ₹80-150
- Bulk shipping (10kg+): ₹25-40/kg via rail/freight

### International (USA)
- **Option 1: India Post (Surface)**
  - 500g-2kg: ₹400-800
  - Time: 20-40 days
  - Good for bulk, not urgent

- **Option 2: DHL/FedEx Express**
  - 500g: ₹450-600
  - 1kg: ₹700-900
  - Time: 3-5 days
  - Best for samples/small orders

- **Option 3: ePacket (China/USPS)**
  - Not available from India directly

### Shipping Strategy for Startup
| Order Value | Shipping Free? | Customer Pays |
|-------------|----------------|---------------|
| < $50 | No | $7.99 flat |
| $50-100 | Yes | Free |
| > $100 | Yes | Free |

**Cost to Customer:** Build in $7.99 flat rate for orders under $50

## 4. Returns & Return Shipping

### Industry Standards
- **30-day return window** (common in USA)
- **Return shipping:** Who pays?

### Options for Startup
| Policy | Pros | Cons |
|--------|------|------|
| Customer pays return | Lower cost | Lower conversions |
| Store pays return (first 30 days) | Better UX | Higher costs |
| Store pays (defects only) | Fair | Requires QC |

### Recommended Policy
- **30-day returns** for change of mind
- **Customer pays return shipping** ($6-8)
- **Store pays** if defective/wrong item
- **Free return shipping** for orders over $75

### Return Rate Assumption
- Clothing e-commerce: 15-25% return rate
- Islamic modest wear: Possibly lower (10-15%)
- Budget for: 15% returns = factor into pricing

## 4. Business Entities

### 🇮🇳 India Entity

| Option | Pros | Cons | Cost |
|--------|------|------|------|
| **Sole Proprietorship** | Easy, cheap | No liability protection | ₹2,000-5,000 |
| **Partnership** | Easy for 2+ | Unlimited liability | ₹5,000-10,000 |
| **Private Limited (Pvt Ltd)** | Limited liability, credibility | More compliance | ₹15,000-25,000 |
| **LLP (Limited Liability)** | Flexible, low compliance | Not for large capital | ₹10,000-15,000 |

**Recommended:** **Private Limited** or **LLP**
- Protects personal assets
- Can have foreign direct investment
- Bank loans easier
- **Cost: ₹15,000-25,000 ($175-300)** for registration

### 🇺🇸 USA Entity

| Option | Pros | Cons | Cost |
|--------|------|------|------|
| **Sole Proprietorship** | Simplest, no filing | Personal liability | $0-50/state |
| **LLC (Limited Liability)** | Protected, flexible | Annual fees | $50-800/year |
| **C-Corp** | Can raise VC | Double taxation | $800+/year |
| **S-Corp** | Tax benefits | Strict requirements | $800+/year |

**Recommended for Dropshipping/Small Inventory:**
- **Single-Member LLC** (cheapest, protected)
- Register in home state (e.g., Illinois)
- **Cost: $50-150 (state filing) + $0 (federal for small biz)**

### Structure Recommended
```
Mumin Threads LLC (Illinois, USA)
    ↓ Contract manufacturing
Mumin Threads Pvt Ltd (Tirupur, India)
    ↓ Sourcing & QC
Tirupur Manufacturers
```

## 2. BREAK-EVEN ANALYSIS

### Key Formulas
```
Break-Even Units = Fixed Costs / (Selling Price - Variable Cost per Unit)
Break-Even Revenue = Break-Even Units × Selling Price

Fixed Costs = Platform + Marketing + Operations
Variable Costs = Product Cost + Shipping + Payment Processing + Packaging
```
