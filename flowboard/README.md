# FlowBoard — SaaS Landing Page

A modern, responsive landing page for a fictional SaaS product. Built with pure HTML, CSS, and JavaScript — no frameworks, no build tools.

## Features

- Fully responsive (mobile, tablet, desktop)
- Smooth scroll animations
- Interactive FAQ accordion
- Animated counter stats
- Clean, modern design
- Dark dashboard mockup
- Accessible navigation

## Sections

1. **Navbar** — Fixed, with mobile hamburger menu
2. **Hero** — Headline, CTAs, social proof, dashboard mockup
3. **Features** — 6 feature cards with icons
4. **Pricing** — 3-tier pricing table
5. **Stats** — Animated counters
6. **Testimonials** — Customer quotes
7. **FAQ** — Accordion-style
8. **CTA Banner** — Final call to action
9. **Footer** — Links and social icons

## Tech Stack

- HTML5 (semantic markup)
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## How to Run

1. Clone the repo
2. Open `index.html` in a browser

Or use a local server:
```bash
npx http-server . -p 8080
```

## Preview

The page is fully static — no backend required. Deploy anywhere:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting
